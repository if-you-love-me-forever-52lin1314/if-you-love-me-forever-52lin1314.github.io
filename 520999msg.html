Pop up window, pop-up layer usage：

//Normal pop-up box. Ordinary pop ups only display content, so just pass in the content directly
msg.popups('I am a pop-up window that can input text or HTML');

//Customize pop-up boxes. Custom, a custom attribute is passed in, and the content text is also one of the attributes. The pop-up layer will automatically display in the center.
msg.popups({
	text:'Hallo，520999 created me!',	//The content displayed in the pop-up window supports HTML
	url:'https://github.com/520999js', //Set the URL to open the pop-up window. If both URL and text are set, URL will be used first, The text setting will be invalid
	height:'100px',		//The height displayed by the pop-up layer. The default is auto. Input values such as 100px 10rem, 10%, etc
	width:'90%',		//The width of the pop-up layer display. The default is 90%. Input values such as 100px 10rem, 10%, etc
	close:true,			//Whether to display the close button in the upper right corner, default is true if not passed, display the close button
	background:'#2e2d3c',	//Background color. Hexadecimal color coding. The default is' # 2e2d3c '
	opacity:92,			//The transparency of the pop-up layer is set to 92 by default, with values ranging from 0 to 100. 0 indicates opacity, while 100 indicates full transparency
	padding:'3px',		//The gap left around the pop-up layer is 1rem by default. Can be transmitted as 10px 1rem, etc
});


//Confirmation prompt for information confirmation and cancellation (to be improved, only initially added)
msg.confirm({
	text:'hallo',
	buttons:{
		'确认':function(){
			alert('点了确认');
		},
		'取消':function(){
			alert('点了取消');
		}
	} 
});
